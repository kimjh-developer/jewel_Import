## 1. 수입처 판단: 호주 vs 중국

|  | 호주 🇦🇺 | 중국 🇨🇳 |
|--|---------|---------|
| **금** | **적합** — 신고만 하면 무제한 반출 | **부적합** — 사실상 반출 불가 |
| **은** | **적합** — 신고만 하면 무제한 반출 | **부적합** — 2026년 전략물자 지정, 44개 업체만 수출 가능 |
| **CPTPP** | 관세 3% → **0%** | 해당 없음 |
| **결론** | **사업용 수입처로 최적** | **리스크 과다, 사용 불가** |

> 중국발 무허가 은은 한국 통관 시 **밀수입죄** (징역형/고액 벌금) + 수출입 블랙리스트

## 2. 수익성 비교 — 핵심 숫자

| | 은(Silver) 50kg | 금(Gold) 10kg |
|---|---|---|
| **투자금** | 1.3억 원 | 9.5억 원 |
| **매출** | 1.5억 원 | 10억 원 |
| **비용** | ~1,000만 원 | ~500만 원 |
| **VAT 부담** | 200만 원 | **0원** (KRX) |
| **수수료** | 5% (민간) | 0.3~0.5% (KRX) |
| **영업이익** | **1,000만 원** | **4,500만 원** |
| **수익률** | **6.7%** | **4.7%** |

**판단 기준:**
- **소자본(1~2억)** → 은 중심 (수익률 높음, 진입 용이)
- **대자본(10억+)** → 금 중심 (절대 수익 4.5배, VAT 면제, 수수료 1/10)
- **최적** → 금·은 병행으로 리스크 분산

## 3. 금이 은보다 유리한 3가지 이유

| 구분 | 금 (KRX 경유) | 은 (민간 거래소) |
|------|--------------|----------------|
| **부가세** | 면제 (0%) | 10% 부과 |
| **거래 수수료** | 0.3~0.5% | 5% |
| **물류 효율** | 10kg = 주먹 크기 | 50kg = 대형 상자, 보관료 2배 |

> KRX 금시장이 없는 은은 민간 거래소를 통해야 하므로, 동일 마진이라도 금의 실질 수익이 10%+ 높음

| 순서 | 할 일 | 기관 |
|------|-------|------|
| 1 | 법인 설립 (정관에 '귀금속 수출입 및 도소매' 명시) | 등기소 |

## 5. FIU(금융정보분석원) 핵심 의무 요약

### 반드시 해야 할 4가지

| 의무 | 내용 | 위반 시 |
|------|------|---------|
| **보고기관 등록** | KoFIU 시스템 가입 + 준법감시인 지정 | 과태료 2,000만 원 |
| **고객확인(CDD)** | 모든 거래 상대 신원 + **자금 출처** 확인·기록 | 과태료 1,000만 원 |
| **의심거래보고(STR)** | 의심 거래 인지 후 **3영업일** 내 보고 | 과태료 3,000만 원 + 형사처벌 |
| **고액현금거래보고(CTR)** | 현금 **1,000만 원 이상** 거래 시 30일 내 보고 | 과태료 2,000만 원 |

### 자금 출처 확인 — 사업자의 핵심 의무

- **법적 근거:** 특금법 제5조의2 — 귀금속 사업자는 고객의 **자금 원천(Source of Funds)** 확인 의무
- 확인 거부 시 → **거래 거절 권한** 있음 (확인 없이 거래 시 사업자가 법적 책임)
- 기록 보존: **5년간 의무**
- 자금세탁에 이용 시 → **사업자도 공범/방조범 처벌 가능**

### 실무 팁

> **현금 거래를 일체 받지 않고 계좌이체만 허용**하면 CTR 보고 의무가 사업자에게 발생하지 않음 (은행이 처리). FIU 리스크를 가장 효과적으로 줄이는 방법.

## 6. 판매 루트별 절차 요약

### 금 → KRX 금시장 (5단계)

```
증권사 법인계좌 개설 → 품위검사(99.99%) → KRX 입고(예탁결제원)
→ 시장 매도(HTS/MTS) → T+2 정산
```

### 은 → 한국금거래소 위탁 (5단계)

```
B2B 상담 → 품위검사(99.9%) + 재주조 → 위탁계약 → 금고 입고
→ 판매 완료 후 정산
```
