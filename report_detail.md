## 2. FTA/CPTPP 혜택 분석

### 2-1. 관세 절감 효과

| 항목 | FTA/CPTPP 미적용 | FTA/CPTPP 적용 | 비고 |
|------|------------------|----------------|------|
| 은 기본 관세 | **3%** | **0%** | HS Code: 7106.91 |
| 금 기본 관세 | **3%** | **0%** | HS Code: 7108.12 (괴·바) |
| 은 50kg(1.5억 원) 절감액 | - | **약 450만 원** | |
| 금 10kg(10억 원) 절감액 | - | **약 3,000만 원** | 금은 고가이므로 절감 효과 극대화 |

> **금은 가치 대비 관세 절감 효과가 은보다 훨씬 크므로, FTA/CPTPP 적용이 사업 수익성의 핵심**

### 2-2. CPTPP 추가 장점
- 여러 회원국 간 **누적 원산지 인정** (호주산 귀금속을 뉴질랜드에서 가공해도 역내산 인정)
- 통관 간소화 및 우선 통관 혜택
- 물류 시간 단축

### 2-3. 적용 요건
- 호주 공급처로부터 **CPTPP 원산지 증명서(C/O)** 발급 필수
- 은: HS Code **7106.91** / 금: HS Code **7108.12** (괴·바 기준) 확인 필요

## 3. 법인 설립 및 준법 사항 (FIU 대응)

### 3-1. 사업자 등록 필수 요건

| 항목 | 요구 사항 |
|------|-----------|
| 업종 코드 | 귀금속 및 관련 제품 도매업(46492) |
| 정관 명시 | '귀금속 수출입 및 도소매' (금·은 모두 포함되도록 기재) |
| 자본금 | 최소 5,000만 원 ~ 1억 원 권장 (금 취급 시 거래 단가가 높으므로 **1억 원 이상 권장**) |
| 사업장 | 실제 사업장 필수 (비상주 사무실 불가) |
| 통관 고유부호 | 관세청 발급 필수 |

### 3-2. 금융정보분석원(FIU) 등록 및 신고 제도 상세

귀금속 판매업자는 특정금융정보법(특금법)에 의해 **'특정비금융사업자(DNFBP)'**로 분류되며, FIU의 자금세탁방지(AML) 체계에 의무적으로 편입됩니다. 이 섹션에서는 시스템 가입부터 신고 절차까지 전 과정을 다룹니다.

#### 3-2-1. FIU 보고 시스템(KoFIU) 가입 절차

```
[1단계] 사업자 등록 완료
   └ 국세청에 '귀금속 도소매업' 사업자 등록 완료 상태여야 함

[2단계] 금융정보분석원 홈페이지 접속
   └ https://www.kofiu.go.kr → '보고기관 등록' 메뉴

[3단계] 보고기관 등록 신청
   └ 필요 서류:
     · 사업자등록증 사본
     · 법인등기부등본 (법인 사업자)
     · 대표자 신분증 사본
     · 자금세탁방지 내부통제 규정서 (자체 작성)
     · 보고책임자(준법감시인) 지정서

[4단계] 보고 시스템 접근 권한 발급
   └ FIU에서 심사 후 '보고기관 코드' 및 시스템 접근 ID/PW 발급
   └ 소요 기간: 약 2~4주

[5단계] 보고 담당자 교육 이수
   └ FIU에서 제공하는 온라인 AML 교육 이수 필수
   └ 연 1회 이상 정기 교육 참여 의무
```

#### 3-2-2. 내부통제 체계 구축 (사업 개시 전 필수)

| 구축 항목 | 세부 내용 | 비고 |
|-----------|-----------|------|
| **준법감시인(보고책임자) 지정** | 대표이사 또는 임원급 1인을 AML 보고책임자로 지정 | 소규모 법인은 대표 겸임 가능 |
| **내부통제 규정 마련** | 고객 확인, 거래 모니터링, 의심거래 판단 기준 등을 문서화 | FIU 등록 시 제출 필수 |
| **거래 기록 보존** | 모든 귀금속 거래 기록을 **5년간 보존** | 거래일자, 금액, 상대방 정보 포함 |
| **직원 교육** | AML 관련 내부 교육 연 1회 이상 실시 | 교육 이수 기록 보관 |
| **독립적 감사** | 내부통제 이행 여부를 주기적으로 점검 | 외부 감사 병행 권장 |

#### 3-2-3. 고객확인제도(CDD/EDD) — 거래 전 필수 절차

귀금속을 매수하거나 위탁하는 모든 상대방에 대해 신원 확인을 수행해야 합니다.

**일반 고객확인(CDD: Customer Due Diligence)**

| 확인 항목    | 개인 고객               | 법인 고객                 |
| -------- | ------------------- | --------------------- |
| 신원 확인    | 실명, 주민등록번호, 주소, 연락처 | 법인명, 사업자등록번호, 대표자 정보  |
| 신분증 확인   | 주민등록증 / 운전면허증 / 여권  | 사업자등록증 + 대표자 신분증      |
| 거래 목적 확인 | 투자, 선물, 장신구 제작 등    | 원자재 매입, 재판매 등         |
| 실소유자 확인  | 본인 여부 확인            | 최종 실소유자(지분 25% 이상) 확인 |

**강화된 고객확인(EDD: Enhanced Due Diligence)** — 아래 경우 적용

| EDD 적용 대상 | 추가 확인 사항 |
|---------------|---------------|
| 고위험 국가 거주자 (FATF 지정) | 자금 출처 증빙, 거래 목적 심층 확인 |
| 정치적 주요인물(PEP) | 자산 및 소득 출처, 사회적 지위 확인 |
| 대규모 현금 거래자 | 자금 출처 증빙 서류 요구 |
| 의심 거래 이력자 | 과거 거래 패턴 분석, 추가 서류 요구 |
| **금 1kg 이상 단일 거래** | 자금 출처, 구매 목적, 보관 계획 확인 권장 |

#### 3-2-3-1. 자금 출처 확인 및 기록 의무 (귀금속 사업자의 핵심 의무)

귀금속은 현금화가 용이하여 자금세탁의 주요 수단으로 악용될 수 있습니다. 따라서 특정금융정보법 제5조의2에 의거, 귀금속 사업자는 **모든 고객의 자금 출처를 확인하고 기록·보존할 법적 의무**가 있습니다.

**자금 출처 확인이 필수인 경우**

| 거래 유형 | 기준 금액 | 확인 수준 |
|-----------|-----------|-----------|
| 일반 현금 거래 | **1,000만 원 이상** | 자금 출처 확인 + CTR 보고 |
| 계좌이체 거래 | **2,000만 원 이상** (권장) | 자금 출처 확인 (사업자 자체 기준) |
| 고위험 고객 거래 | **금액 무관** | 자금 출처 증빙 서류 요구 필수 |
| 금 거래 (고가) | **500만 원 이상** (권장) | 자금 출처 + 거래 목적 확인 |
| 반복 거래 고객 | **누적 5,000만 원 이상** (권장) | 정기적 자금 출처 재확인 |

> **법적 근거:** 특금법 제5조의2(고객확인의무)에 따라, 귀금속 사업자(특정비금융사업자)는 거래 상대방의 신원뿐 아니라 **자금의 원천(Source of Funds)**을 합리적으로 확인해야 합니다. 이를 이행하지 않으면 과태료 및 FIU 고위험 사업자 지정 대상이 됩니다.

**자금 출처 확인 방법 및 증빙 서류**

| 자금 출처 유형 | 요청 가능한 증빙 서류 | 비고 |
|----------------|----------------------|------|
| 급여/사업소득 | 근로소득 원천징수영수증, 소득금액증명원, 사업자 재무제표 | 가장 일반적 |
| 부동산 매각 | 부동산 매매계약서, 등기이전 확인서 | 대금 수령 시점 확인 |
| 금융 자산 매각 | 증권사 매도 확인서, 예금 해지 증명서 | 해당 금융기관 발행 |
| 상속/증여 | 상속세/증여세 납부 확인서, 법원 상속 결정문 | 세무서 발행 |
| 대출금 | 금융기관 대출 실행 확인서 | 대출 목적 일치 여부 확인 |
| 해외 송금 | 외국환 매입 증명서, 해외 소득 증빙 | 외환거래법 준수 여부 동시 확인 |
| 보험금/퇴직금 | 보험금 지급 확인서, 퇴직금 정산서 | 지급 기관 확인 |

**자금 출처 확인 실무 절차**

```
[1단계] 거래 접수 시 — 고객에게 자금 출처 질문
   └ "이번 구매 자금은 어디에서 마련하셨나요?" (구두 확인)
   └ 거래 금액, 고객 유형에 따라 증빙 서류 요청 여부 판단

[2단계] 증빙 서류 수령 및 검증
   └ 고객이 제출한 서류의 진위 확인 (발급 기관, 날짜, 금액 일치)
   └ 서류상 금액과 거래 금액의 합리적 일치 여부 확인
   └ 불일치 시 추가 설명 또는 보완 서류 요청

[3단계] 기록 작성 및 보관
   └ '자금 출처 확인서' 양식에 내용 기재 (자체 양식 마련 필수)
   └ 고객 신원 정보 + 자금 출처 유형 + 증빙 서류 사본 첨부
   └ **보관 기간: 거래 종료 후 5년간 의무 보존**

[4단계] 이상 징후 발견 시 — STR 보고 연계
   └ 자금 출처 설명이 불합리하거나 증빙 거부 시
   └ 준법감시인에게 보고 → STR 보고 여부 판단
```

**자금 출처 확인을 거부하는 고객 대응**

| 상황 | 사업자의 대응 | 법적 근거 |
|------|-------------|-----------|
| 고객이 자금 출처 설명 거부 | **거래 거절 가능** (거절 권한 있음) | 특금법 제5조의2 제4항 |
| 증빙 서류 제출 거부 | **거래 거절 권장** + STR 보고 검토 | 특금법 제5조의2 제4항 |
| 허위 서류 제출 의심 | **거래 즉시 중단** + STR 보고 | 특금법 제4조 |
| 대리인 거래 (본인 미출석) | 대리인 신원 + 본인 위임장 + 자금 출처 모두 확인 | 특금법 시행령 |

> **실무 핵심:** 고객이 자금 출처 확인을 거부할 경우, 사업자는 **거래를 거절할 법적 권리**가 있으며 오히려 확인 없이 거래를 진행하면 **사업자가 법적 책임**을 지게 됩니다. "고객 불편"보다 "법적 준수"가 우선입니다.

**귀금속 사업에서 자금 출처 확인이 특히 중요한 이유**

| 이유 | 설명 |
|------|------|
| 현금 세탁 위험 | 불법 자금 → 금·은 구매 → 되팔아 합법 자금으로 전환하는 루트가 일반적 |
| 거래 단가 높음 | 금 1kg = 약 1억 원, 소량 거래에도 고액 현금이 오감 |
| 익명성 유혹 | 귀금속은 실물이라 추적이 어렵다는 인식 → 자금세탁 시도 빈번 |
| FIU 집중 감시 대상 | 귀금속업은 FATF 권고에 의해 전 세계적으로 고위험 업종 분류 |
| 사업자 연대 책임 | 자금세탁에 이용된 사실이 밝혀지면, **사업자도 공범 또는 방조범**으로 처벌 가능 |

> **결론:** 귀금속 사업자는 단순한 "판매자"가 아니라 **"금융 게이트키퍼(Gatekeeper)"** 역할을 수행해야 합니다. 자금 출처 확인은 선택이 아닌 **생존 조건**입니다. 특히 금은 1건당 거래액이 수천만~수억 원이므로, 모든 거래에 대해 자금 출처 확인을 기본 정책으로 수립하시기 바랍니다.

#### 3-2-4. 의심거래보고(STR) 절차 상세

**STR(Suspicious Transaction Report):** 자금세탁 또는 테러자금조달이 의심되는 거래를 FIU에 보고하는 제도

**신고 대상 (귀금속 사업 관련 주요 유형)**

| 의심 유형 | 구체적 사례 |
|-----------|------------|
| 분할 거래 | 고객이 CTR 기준(1,000만 원)을 회피하기 위해 여러 번에 나눠 구매 |
| 신원 불일치 | 제3자 명의 계좌로 입금하거나, 타인 신분증 제시 |
| 비정상 거래 패턴 | 매입 후 단기간 내 전량 매도 반복, 손실 감수 매도 |
| 고액 현금 지참 | 수천만 원대 현금을 직접 가져와 구매 시도 |
| 출처 불명 | 자금 출처 설명 거부, 소득 수준과 맞지 않는 대량 구매 |
| 해외 송금 연계 | 귀금속 구매 직후 해외 반출 시도 |

**STR 보고 절차**

```
[1단계] 의심 거래 감지
   └ 직원이 거래 과정에서 이상 징후 발견
   └ 또는 내부 모니터링 시스템에서 자동 감지

[2단계] 내부 보고
   └ 발견 직원 → 준법감시인(보고책임자)에게 즉시 보고
   └ 준법감시인이 보고 여부 최종 판단

[3단계] FIU 온라인 보고
   └ KoFIU 보고 시스템(https://www.kofiu.go.kr) 접속
   └ STR 보고서 작성 및 제출
   └ 보고 기한: 의심 거래 인지 후 **3영업일 이내**

[4단계] 보고 내용 기밀 유지 (Tipping-off 금지)
   └ 고객에게 STR 보고 사실을 절대 알려서는 안 됨
   └ 위반 시 형사처벌 대상
```

> **주의:** STR 보고는 '확증'이 아닌 '의심'만으로도 보고 의무가 발생합니다. 보고하지 않으면 과태료(최대 3,000만 원) 및 형사처벌 가능

#### 3-2-5. 고액현금거래보고(CTR) 절차 상세

**CTR(Currency Transaction Report):** 일정 금액 이상의 현금 거래를 FIU에 자동 보고하는 제도

| 항목 | 내용 |
|------|------|
| 보고 기준 | **1회 1,000만 원 이상** 현금(지폐·수표) 수수 시 |
| 보고 주체 | 귀금속 판매 사업자 (특정비금융사업자) |
| 보고 기한 | 거래일로부터 **30일 이내** |
| 보고 방법 | KoFIU 온라인 보고 시스템에서 CTR 양식 작성 및 제출 |
| 보고 내용 | 거래일시, 거래금액, 현금 종류, 고객 신원 정보, 거래 유형 |

**CTR 보고 절차**

```
[1단계] 현금 거래 발생
   └ 고객이 1,000만 원 이상의 현금으로 금·은 구매 또는 판매 대금 수령

[2단계] 거래 기록 즉시 작성
   └ 거래일시, 금액, 고객 신원 정보, 거래 목적 기록
   └ 고객의 신분증 사본 확보 필수

[3단계] FIU 온라인 보고
   └ KoFIU 보고 시스템 접속 → CTR 양식 선택
   └ 거래 정보 입력 및 제출
   └ 보고 기한: 거래일로부터 30일 이내

[4단계] 기록 보존
   └ CTR 보고 기록 및 관련 증빙을 5년간 보관
```

**귀금속 사업에서 CTR 발생 시나리오**

| 시나리오 | CTR 해당 여부 | 비고 |
|----------|--------------|------|
| 고객이 현금 1,500만 원으로 은괴 구매 | **해당** | 즉시 기록 + 30일 내 보고 |
| 고객이 계좌이체 5,000만 원으로 금괴 구매 | **비해당** | 은행이 별도 보고 (사업자 의무 없음) |
| 고객이 현금 800만 원 + 이체 500만 원 | **비해당** | 현금 부분만 1,000만 원 미만 |
| 고객이 오전 700만 원 + 오후 500만 원 현금 구매 | **해당 의심** | 분할 거래 의심 → STR 보고도 고려 |

> **실무 팁:** 귀금속 사업에서 가장 안전한 방법은 **현금 거래를 일체 받지 않고 계좌이체만 허용**하는 것입니다. 이 경우 CTR 보고 의무가 사업자에게 발생하지 않으며(은행이 처리), FIU 리스크를 대폭 줄일 수 있습니다.

#### 3-2-6. FIU 미이행 시 제재

| 위반 유형 | 제재 내용 |
|-----------|-----------|
| 보고기관 미등록 | 과태료 최대 **2,000만 원** |
| STR 미보고 | 과태료 최대 **3,000만 원** + 형사처벌 가능 |
| CTR 미보고 | 과태료 최대 **2,000만 원** |
| 고객확인(CDD) 미이행 | 과태료 최대 **1,000만 원** |
| 기록 미보존 (5년) | 과태료 최대 **1,000만 원** |
| Tipping-off (STR 보고 사실 누설) | **형사처벌** (2년 이하 징역 또는 1,000만 원 이하 벌금) |
| 내부통제 규정 미마련 | 시정명령 + 과태료 |

> **경고:** FIU 의무를 반복적으로 위반하면 '고위험 사업자'로 지정되어 **금융기관 거래 제한**(계좌 동결, 대출 중단 등) 및 **수출입 제한** 조치를 받을 수 있습니다. 귀금속 사업의 존속 자체가 위협받는 수준의 제재입니다.

#### 3-2-7. 금 vs 은 거래 시 FIU 신고 빈도 비교

| 구분 | 은(Silver) | 금(Gold) |
|------|------------|----------|
| 1kg 현재 시세 | 약 300~350만 원 | 약 1억 원 |
| CTR 기준(1,000만 원) 초과 물량 | 약 3kg 이상 | **약 10g 이상** |
| 현금 거래 시 CTR 발생 빈도 | 상대적으로 낮음 | **거의 모든 거래에서 발생** |
| EDD 적용 가능성 | 대량 거래 시 | **소량 거래에서도 적용 가능** |

> **금 사업의 FIU 핵심:** 금은 10g(약 100만 원)만 거래해도 소액이지만, 사업 규모가 커지면 1건당 거래액이 수천만~수억 원에 달합니다. **현금 거래를 원천 차단하고, 계좌이체 전용 정책을 수립**하는 것이 FIU 리스크 관리의 핵심입니다.

### 3-3. 외국환거래법 준수
- 수입 대금은 반드시 **외국환은행 지정 후 법인 계좌**를 통해 송금
- 수입신고필증과 물품매매계약서(Invoice)의 **일치** 필수
- 개인 계좌 또는 핸드캐리 대금 지급 시 **외국환거래법 위반** (검찰 조사 대상)
- 금의 경우 거래 금액이 크므로 **은행의 외환 송금 한도 사전 협의** 필수

### 3-4. 법인 설립 체크리스트

| 단계 | 필수 절차 | 관련 기관 |
|------|-----------|-----------|
| 법인 설립 | 정관에 사업 목적 명시, 적정 자본금 확보 | 등기소 |
| 사업자 등록 | 정확한 업종 코드, 실제 사업장 | 국세청 |
| 전담 은행 지정 | 기업 외환 계좌 개설 | 시중 은행 |
| FIU 시스템 등록 | KoFIU 보고 시스템 가입 | 금융정보분석원 |
| 관세청 등록 | 통관 고유부호 발급 | 관세청 |

## 4. 한국 내 판매 채널 비교: 금 vs 은

### 4-1. 금(Gold)의 판매 채널 — KRX 금시장의 압도적 장점

| 채널 | 특징 | 부가세(VAT) | 적합도 |
|------|------|-------------|--------|
| **KRX 금시장** | 한국거래소 운영 공식 현물시장 | **면제 (0%)** | **최적** |
| 민간 거래소 (한국금거래소 등) | 실물 금 매매, 위탁판매 | 10% 부과 | 보통 |
| 골드뱅킹 (은행) | 계좌형 투자 | 매매차익 15.4% | 투자용 |

> **핵심:** 금은 KRX 금시장을 통해 판매하면 **부가세 10%가 면제**되므로, 민간 거래소 대비 수익성이 크게 향상됩니다. 이것이 금 사업의 가장 큰 장점입니다.

### 4-2. 은(Silver)의 판매 채널 — KRX 은시장 미개설

| 채널 | 특징 | 부가세(VAT) | 적합도 |
|------|------|-------------|--------|
| KRX 금시장 | **은 시장 미개설** | - | 불가 |
| **민간 거래소 (한국금거래소 등)** | 실물 은 매매, 위탁판매 | **10% 부과** | **현실적 최선** |
| 실버뱅킹 (신한은행 등) | 계좌형, 0.01g 단위 소액 투자 | 매매차익 15.4% | 소액 투자용 |
| ETF/ETN (KODEX 은선물 등) | 주식 계좌에서 실시간 거래 | 매매차익 15.4% | 시세 차익용 |
| RWA 토큰화 플랫폼 | 실물 기반 디지털 인증서 거래 | 플랫폼별 상이 | 조각 투자/STO |

> **KRX 은 시장이 없는 이유:** 부가가치세 면제 혜택이 은에는 적용되지 않음 (금은 양성화 정책으로 면세)

### 4-3. 금 vs 은 세금 구조 핵심 차이

| 구분 | 금(Gold) — KRX 경유 | 은(Silver) — 민간 거래소 |
|------|---------------------|------------------------|
| 매입 시 VAT | **면제** | 10% (매입세액 공제 가능) |
| 매출 시 VAT | **면제** | 10% (소비자 징수) |
| VAT 실질 부담 | **0원** | 매출-매입 차액분 납부 |
| 양도소득세 | 비과세 (개인, KRX 경유) | 해당 없음 (사업소득으로 과세) |

> **금은 KRX 경유 시 부가세가 완전 면제되어, 같은 마진율이라도 은보다 실질 수익이 10% 이상 높아질 수 있습니다.**

## 5. 한국금거래소/KRX 위탁판매 절차

### 5-1. 은(Silver) — 민간 거래소 위탁판매 (5단계)

```
[Step 1] 위탁 상담 및 사업자 확인
   └ 법인 사업자등록증 + 수입신고필증 지참 → B2B 파트너십 상담

[Step 2] 품위 및 진위 검사 (Assay)
   └ 순도 99.9% 이상 확인 → 국내 유통용 재검인(Hallmark) 또는 재주조

[Step 3] 위탁 계약 체결
   └ 수수료, 보관료, 보험, 판매 기간 등 명시

[Step 4] 입고 및 전산 등록
   └ 거래소 금고 입고 → 입고 확인서(보관증) 발급

[Step 5] 판매 및 정산
   └ 판매 완료 → 수수료 공제 후 법인 계좌 입금 + 세금계산서 발행
```

### 5-2. 금(Gold) — KRX 금시장 경유 판매 절차

```
[Step 1] KRX 금시장 회원사(증권사) 계좌 개설
   └ KRX 금시장 거래가 가능한 증권사에 법인 계좌 개설

[Step 2] 품위 검사 및 인증
   └ KRX 지정 검정기관에서 순도 99.99% 이상 확인
   └ LBMA 인증 브랜드(Perth Mint 등)는 간소화 가능

[Step 3] KRX 금시장 입고
   └ KRX 지정 보관기관(한국예탁결제원 등)에 실물 금 입고
   └ 금 현물 계좌에 수량 반영

[Step 4] 시장 매도 주문
   └ 증권사 HTS/MTS를 통해 KRX 금시장에서 실시간 매도
   └ 1g 단위 거래 가능

[Step 5] 정산
   └ 매도 체결 후 T+2일 정산 → 증권 계좌 입금
   └ 부가세 면제, 양도소득세 비과세(개인)
```

> **금은 KRX를 통해 직접 시장에서 매도할 수 있어, 민간 거래소 위탁판매 수수료(5%)보다 훨씬 저렴한 증권 거래 수수료(0.3~0.5%)로 판매 가능**

### 5-3. 재주조(Re-casting) 분석

| 항목 | 그대로 판매 | 국내 브랜드 재주조 후 판매 |
|------|-------------|--------------------------|
| 추가 비용 (은 50kg) | 0원 | 약 150~200만 원 |
| 추가 비용 (금 10kg) | 0원 | 약 200~300만 원 |
| 판매 난이도 | 어려움 (매수자 확보 힘듦) | 매우 쉬움 (즉시 판매 가능) |
| 마진율 | 낮음 (감가 적용) | 높음 (브랜드 프리미엄) |

**재주조 비용 세부 (은 50kg):**
- 정련 및 공임: 약 80~120만 원
- 분석 및 감정료: 약 20~30만 원
- 해리(Melt Loss): 0.2%~0.5% (약 100g~250g 소실, 시세 30~75만 원 상당)

### 7-2. 전략적 권고

1. **수입처 선정:** 호주 적극 권장 (CPTPP 관세 0%, 규제 투명). 중국은 금·은 모두 부적합
2. **공급처:** LBMA 인증 업체 우선 (Perth Mint, ABC Bullion 등) → 재주조 비용 절감 + KRX 입고 간소화
3. **금·은 병행 전략:** 금(KRX, 고마진·저수수료)과 은(민간 거래소, 소자본 고회전)을 병행하여 리스크 분산
4. **금 우선 고려:** 자본 여력이 있다면 KRX 금시장의 VAT 면제 + 저수수료 이점을 활용한 금 사업이 수익성 면에서 압도적
5. **전문가 자문:** 관세사(통관/FTA) + 세무사(귀금속 특화) 필수 자문
6. **자금 투명성:** 전자세금계산서 100% 발행, 법인 계좌 거래만 유지
